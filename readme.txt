This is the user readme for the cadesim client.

To run:
* You will need Java 8 (recommend using 32-bit for PP compatibility!). https://www.java.com/en/download/
* unzip the cadesim files somewhere sensible and double-click the .jar to launch
* Contact the server admin for the server code
